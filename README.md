# 💎 My Gemini Gems

Uma coleção de assistentes personalizados (Gems) criados para o Google Gemini, projetados para otimizar tarefas diárias, desenvolvimento de software e criação de conteúdo.

Este repositório centraliza os links de acesso direto e as diretrizes de configuração de cada agente.

## 🚀 Como Criar e Usar estas Gems

Este repositório foi estruturado para que você possa replicar facilmente qualquer uma das Gems no seu painel do Google Gemini. 

Cada arquivo `.md` dentro da pasta `gems/` representa uma Gem individual:
*   **Nome do Arquivo:** É o título exato que você deve dar à sua Gem.
*   **Conteúdo do Arquivo:** É o prompt completo que você deve copiar e colar no campo de instruções.

### 🛠️ Passo a Passo para Configuração:

1. **Acesse o Gemini:** Abra o [Google Gemini](https://gemini.google.com) no seu navegador.
2. **Abra o Menu de Gems:** No menu lateral esquerdo, clique em **Gems** e selecione a opção **"Criar uma Gem"**.
3. **Defina o Nome:** No campo de título, dê à sua Gem exatamente o mesmo nome do arquivo Markdown escolhido aqui no repositório (sem a extensão `.md`).
4. **Preencha a Descrição:** Copie o texto da coluna **"Descrição"** da nossa tabela aqui do README e cole no campo correspondente da Gem.
5. **Cole as Instruções:** Acesse o arquivo `.md` correspondente aqui no repositório, copie todo o seu conteúdo em texto e cole diretamente no campo **"Instruções"** da sua Gem.
6. **Finalize:** Escolha um ícone que combine com a função do assistente e clique no botão **"Criar"** no canto inferior direito.

*(Opcional) Se você quiser apenas testar uma Gem antes de criar a sua própria cópia, utilize o link direto disponível na coluna "Testar no Gemini" da tabela abaixo.*

│   │
│   └── Criador de Script para Unificar Markdown.md  # Nome do arquivo = Título da Gem
│                                                    # Conteúdo interno = Prompt da Gem
│
